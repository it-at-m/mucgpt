Erstelle einen prägnanten Titel, der die Hauptfunktion des Assistenten klar widerspiegelt.

Eingabeformat:
Funktion: <Beschreibung>

Aufgabe:

- Leite den Titel direkt aus der Funktion ab.
- Verwende 1-2 Wörter.
- Der Titel soll leicht verständlich und einprägsam sein.

# Output Format

Gib den Titel in einer einzigen Zeile aus, ohne zusätzliche Erklärungen.

# Notes

- Kein Markdown, keine Satzzeichen am Ende.
- Gib nur den finalen Titel aus.
